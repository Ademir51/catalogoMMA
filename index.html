Projeto

Nosso site, da matéria de desenvolvimento web realizado pela equipe MMA. Trará foco nos detalhes das atividades concluídas, próximos passos a serem seguidos e decisões tomadas. Descrição do projeto Esse projeto web tem como objetivo criar e desenvolver uma página web interativa e informativa sobre o futebol, com foco nos times paulistas e na seleção brasileira. Está página servirá para levar informações atualizadas e envolventes ao usuário, bem como demonstrar as habilidades aprendidas em desenvolvimento web, incluindo CSS, HTML5 e JavaScript A página terá a proposta de trazer praticidade e agilidade na navegação do usuário, mas sem deixar a qualidade de lado. O projeto também contará com um desing responsivo para garantir melhor experiência e acessibilidade aos clientes independete do dispositivo que for usado, como notebooks e celulares. Além disso, este projeto web pode conter recursos adicionais, como: -Tela de login -Galeria de imagens O desenvolvimento da página web será conduzido de acordo com as melhores práticas de desenvolvimento web, e a equipe se esforçará para entregar um produto final que atenda aos requisitos estabelecidos. Este projeto é uma oportunidade para aplicar os conhecimentos adquiridos na disciplina de Desenvolvimento Web e criar uma presença online significativa relacionada ao tema escolhido. Espera-se que a página web seja informativa, atraente e funcional, proporcionando uma experiência positiva aos visitantes. “Esta breve descrição do projeto web fornece uma visão geral do que o projeto visa alcançar e o foco nas habilidades de desenvolvimento web a serem aplicadas. Os alunos podem personalizar esta descrição de acordo com o tema específico do projeto que estão desenvolvendo.”
